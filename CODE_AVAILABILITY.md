# Code availability statement

All custom R, Python, Jupyter Notebook, and shell scripts used for single-cell RNA-sequencing preprocessing, RNA-velocity analysis, trajectory inference, cell-fate analysis, metabolite-mediated communication analysis, transcription-factor motif analysis, and figure generation are available at **[GitHub repository URL]**. The repository includes environment specifications, installation instructions, a demonstration-data specification, expected outputs, and a figure-to-script mapping. Raw and processed sequencing data are available as described in the Data availability statement.
