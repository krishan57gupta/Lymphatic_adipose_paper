# Demo dataset specification

Add a small shareable dataset that runs on a normal desktop. Recommended contents:

- `demo_counts.mtx.gz`
- `demo_genes.tsv.gz`
- `demo_barcodes.tsv.gz`
- `demo_metadata.csv`
- optionally, a small demo loom or h5ad object

The demo should test loading, basic QC/normalization, a small visualization, and one exported numerical result. State clearly that the demo is for software validation and is not intended to reproduce biological conclusions.
