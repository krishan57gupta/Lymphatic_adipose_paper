# Nature code and software checklist mapping

| Nature requirement | Repository location | Status |
|---|---|---|
| Source code | `scripts/` | Included |
| Compiled standalone software | Not applicable unless authors claim a standalone tool | N/A |
| Small demo dataset | `data/demo/` | Must be added |
| System requirements | `README.md`, `docs/TESTED_SYSTEMS.md` | Template included |
| Dependencies and versions | `README.md`, `environment/` | Draft included; export exact environments |
| Tested versions | `docs/TESTED_SYSTEMS.md` | Must be completed |
| Non-standard hardware | `README.md`, `docs/TESTED_SYSTEMS.md` | Must be completed |
| Installation guide | `README.md` | Included |
| Typical installation time | `docs/TESTED_SYSTEMS.md` | Must be measured |
| Demo instructions | `data/demo/README.md` | Specification included |
| Expected output | `expected_output/demo/` | Must be added after demo |
| Demo run time | `docs/TESTED_SYSTEMS.md` | Must be measured |
| Instructions for own data | `README.md`, `config/` | Included at high level |
| Reproduce quantitative results | `FIGURE_SCRIPT_MAP.tsv` | Must be completed |
| License | `LICENSE` | Included; institutional review needed |
| Open repository link | `CITATION.cff`, `CODE_AVAILABILITY.md` | Add after GitHub/Zenodo release |
| Code functionality/pseudocode in manuscript | Manuscript Methods | Authors must confirm |
